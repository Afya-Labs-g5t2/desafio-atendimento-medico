Rotas

As rotas da Api são:

  • /enderecos
  • /pacientes
  • /profissoes
  • /especialistas (em breve)
  • /agendamentos (em breve)

GET


/enderecos

retorna uma lista só com endereços

/enderecos/id

retorna o endereços de id específico e com a lista de pacientes que moram no mesmo endereço

/pacientes

retorna uma lista só com pacientes

/pacientes/id

retorna o paciente de id específico e com seu respectivo endereço

/profissoes

retorna uma lista só com profissoes

/profissoes/id

retorna a profissão de id específico

POST


/enderecos

Cria um novo endereço através de um body json:

{
"cep":"36000-011",
"logradouro":"Rua dos bobos",
"numero":"0",
"bairro":"A casa",
"cidade":"Galinha pintadinha",
"uf":"MG"
}

/pacientes

Cria um novo paciente associado a um endereço existente como no exemplo abaixo:

{
"cpf": "951",
"nome": "renato",
"data_nascimento": "1987-06-30",
"telefone": "3421 1234",
"celular": "999 888 777",
"email": "email@gmail.com",
"tipo_sangue": "O-",
"id_endereco":"2"
}

/profissoes

Cria uma nova profissão através de um body json:

{
"profissao":"Dentista"
}

PUT


/enderecos/id

Update em um endereço de id específico

/pacientes/id

Update em um paciente de id específico

/profissoes/id

Update em uma profissão de id específico

DELETE


/enderecos/id

deleta um endereço de id específico

/pacientes/id

deleta um paciente de id específico

/profissoes/id

deleta uma profissão de id específico
