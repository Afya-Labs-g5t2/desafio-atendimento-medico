Prontuário Digital

Uma API desenvolvida em node.js, Sequelize e Postgres para o desafio da Gama Academy em conjunto com a Afya Educacional

Bora começar?!

Rotas

As rotas da Api são:

  • /enderecos
  • /pacientes
  • /profissoes
  • /especialistas
  • /agendamentos (em breve)

GET


/enderecos

retorna uma lista só com endereços

/enderecos/id

retorna o endereços de id específico e com a lista de pacientes e especialistas que moram no mesmo endereço

/pacientes

retorna uma lista só com pacientes

/pacientes/id

retorna o paciente de id específico e com seu respectivo endereço

/profissoes

retorna uma lista só com profissoes

/profissoes/id

retorna a profissão de id específico com a lista de profissionais da área

/especialistas

retorna uma lista só com os especilistas

/especialistas/id

retorna o especialista de id específico com profissão e endereço

POST


/enderecos

Cria um novo endereço através de um body json:

        
                            { 
                                "cep":"36000-011", 
                                "logradouro":"Rua dos bobos", 
                                "numero":"0", 
                                "bairro":"A casa", 
                                "cidade":"Galinha pintadinha", 
                                "uf":"MG" 
                            }

/pacientes

Cria um novo paciente associado a um endereço existente como no exemplo abaixo:

 
                            { 
                                "cpf": "951", 
                                "nome": "renato", 
                                "data_nascimento": "1987-06-30", 
                                "telefone": "3421 1234", 
                                "celular": "999 888 777", 
                                "email": "email@gmail.com", 
                                "tipo_sangue": "O-", 
                                "id_endereco":"2" 
                            }

/profissoes

Cria uma nova profissão através de um body json:

 
                            { 
                                "profissao":"Dentista" 
                            }

/especialistas

Cria um novo especilasita através de um body json:

 
                            {
                                "id": 1,
                                "registro": "194528-SP",
                                "nome": "Aarao Andrade Napoleao Lima",
                                "telefone": "11922234567",
                                "celular": "11922334459",
                                "email": "araoo@example.com",
                                "id_profissao": 4,
                                "id_endereco": 10
                            }
